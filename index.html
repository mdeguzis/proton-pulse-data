About this project

Proton Pulse is 100% open source. Unlike ProtonDB itself, all code, data pipelines, and community configs are public on GitHub. Fork it, self-host it, contribute to it.

Community reports are fetched daily from ProtonDB and stored as static JSON files -- each game gets a latest.json under /data/{appId}/. Users can also submit their own working Proton configurations (version, launch options, environment variables) via the Decky Plugin, which are stored in Supabase and displayed alongside the ProtonDB reports. Reports can be upvoted or downvoted to surface the most helpful compatibility info.

Project repositories:
proton-pulse-data -- Data pipeline, static site, and API.
decky-proton-pulse -- Decky Loader plugin for Steam Deck.
